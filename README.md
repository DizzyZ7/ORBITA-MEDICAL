# ORBITA MEDICAL

Premium interactive website for a coordinated digital clinic.

## Run locally

```bash
npx serve .
```

Open the local URL shown in the terminal. You can also open `index.html` directly, but a local HTTP server is recommended.

Internet access is required for the GSAP and Lenis CDN scripts.

## Structure

- `index.html` — public website markup and content
- `styles.css` — visual system, layouts and responsive behavior
- `scripts/app.js` — navigation, care routes, specialists, forms and motion
- `assets/favicon.svg` — favicon
- `ART_DIRECTION.md` — visual concept
- `CASE_STUDY_RU.md` — Russian case description
- `REELS_SHOT_LIST_RU.md` — recording sequence
- `START_WINDOWS.bat` — Windows launch helper
- `START_MAC_LINUX.sh` — macOS/Linux launch helper
